# Day 01 - Titanic Dataset Exploratory Data Analysis

## Objective

The objective of this activity was to perform basic Exploratory Data Analysis (EDA) on the Titanic dataset and understand the structure and characteristics of the data.

## Dataset

Dataset Used: Titanic Dataset

The dataset contains passenger information such as:

* Passenger ID
* Passenger Class
* Gender
* Age
* Fare
* Survival Status

## Concepts Covered

### Data Loading

* Reading CSV files using Pandas
* Creating a DataFrame

### Dataset Inspection

* Viewing rows using `head()`
* Understanding dimensions using `shape`
* Viewing column names
* Checking data types using `info()`

### Statistical Analysis

* Summary statistics using `describe()`
* Mean
* Minimum and Maximum values
* Standard Deviation

### Missing Value Analysis

* Identifying null values
* Counting missing entries

### Data Visualization

* Count Plot
* Histogram
* Correlation Heatmap

## Libraries Used

* Pandas
* Matplotlib
* Seaborn

## Key Observations

* The dataset contains both numerical and categorical features.
* Some columns contain missing values.
* Survival patterns can be explored using gender and passenger class.
* Correlation analysis helps understand relationships between numerical features.

## Learning Outcomes

After completing this activity, I learned:

* How to load datasets into Python.
* How to inspect and understand a dataset.
* How to identify missing values.
* How to generate basic visualizations.
* How EDA helps in understanding data before building machine learning models.
